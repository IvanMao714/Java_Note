4.2 List集合

有序集合也称为序列

4.2.1 修改并发异常

Iterator<String> it = list.iterator();
while(it.hasNext()){
    String s = it.next();
    if(s.equals("1")){
        it.add("2");
    }
}

因为modCount和expectedModCount会在add后变不同，所以报错

解决方式：

for(int i=0; i<list.size(); i++){
    String s = list.get(i);
    if(s.equals("1")){
        it.add("2");
    }
}
